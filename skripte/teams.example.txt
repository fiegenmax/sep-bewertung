# Format pro Zeile ('#' = Kommentar, Leerzeilen werden ignoriert):
#
#   <kurzname>             Kurzname, mit oder ohne fuehrendes "team-".
#                          Nutzt den Default-Cohort aus .env (GITLAB_COHORT).
#
#   <kurzname> <cohort>    Cohort pro Team ueberschreiben (zwei Tokens),
#                          falls das Team zu einem anderen Tutorium gehoert.
#
#   team-<cohort>-<kurz>   Voller GitLab-Projektname (Cohort wird daraus gelesen).
#
# Daraus entsteht:
#   GitLab-Projekt: {GITLAB_GROUP}/team-{cohort}-{kurzname}
#   lokaler Ordner: team-{kurzname}
#
# Beispiel: "alpha" + GITLAB_COHORT=musterkohorte
#   -> GitLab: .../student_projects/team-musterkohorte-alpha
#   -> lokal:  team-alpha
